Algorithm Summaries

1. Linear Search

Repeat for each element:
    If element == target, return index

2. Binary Search

Split array into half repeatedly until found

3. Bubble Sort

Swap adjacent elements repeatedly until sorted

4. Selection Sort

Find smallest element and place it at start

5. Insertion Sort

Build sorted array one item at a time

6. Fibonacci

Recursively add the last two numbers

7. Factorial

Multiply all numbers down to 1

8. Find Max

Compare each element to track the largest

9. Find Min

Compare each element to track the smallest

10. Sum of Array

Add all elements using loop
